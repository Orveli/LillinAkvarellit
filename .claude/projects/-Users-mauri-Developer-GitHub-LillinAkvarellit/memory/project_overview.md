Kaksi pääosaa:

1. **Interaktiivinen taidekokemus** (`galleria.html`) — fullscreen canvas/WebGL-pohjainen esitys akvarelleista, animaatiot, title overlay, settings-paneeli. Tämä on projektin pääasia.
2. **Kirjan ladontaeditori** (`index.html`) — kuvien järjestely ja hallinta kirjaa varten (save/undo/swap-toiminnot, catalog.json-data).

**Why:** Projekti palvelee kahta tarkoitusta: taiteellinen verkkokokemus ja painotuotteen (kirjan) valmistelu.
**How to apply:** Interaktiivinen taidekokemus on etusijalla. Kirjan ladonta on toissijainen mutta tärkeä osa.
